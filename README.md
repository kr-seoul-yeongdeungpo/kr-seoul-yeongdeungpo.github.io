# 영등포구 · 서울특별시

## 🌐 랜딩 페이지

### **https://kr-seoul-yeongdeungpo.github.io/**

영등포구는 한강 남쪽 서부에 자리 잡은 자치구로, 국회의사당과 63빌딩, 여의도 금융가가 한데 모여 흔히 '한국의 맨해튼'으로 불리는 곳입니다. 구 이름은 음력 이월 초하루부터 보름까지 여의도 샛강변에서 벌어지던 영등굿의 '영등'과 물가 마을을 뜻하는 '포(浦)'가 합쳐진 데서 비롯되었습니다. 방송사 KBS와 증권·선물거래소, 대형 증권사 본사가 밀집해 정치·금융·언론의 심장부 역할을 동시에 수행합니다. 한편 문래동 철공소 골목에는 예술가 공방과 레트로 카페가 스며들어 산업의 옛 정취와 창작 문화가 공존하는 독특한 풍경을 이룹니다. 경방 영등포공장 터에 들어선 타임스퀘어와 봄마다 벚꽃으로 물드는 여의서로가 상징하듯, 영등포는 근대 공업의 유산 위에 현대 도심의 활력을 겹쳐 놓은 자치구입니다.
